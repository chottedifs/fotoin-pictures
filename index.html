Waktu editing bervariasi tergantung paket yang dipilih. Paket Basic selesai dalam 7 hari kerja, Paket Pro 5 hari kerja, dan Paket Premium hanya 3 hari kerja. Kami juga menyediakan layanan express dengan biaya tambahan jika Anda membutuhkan hasil lebih cepat.

Kami menerima pembayaran melalui transfer bank (BCA, Mandiri, BNI), e-wallet (GoPay, OVO, Dana), atau cicilan untuk paket Premium. DP minimal 30% untuk booking tanggal, pelunasan H-3 sebelum acara. Dapatkan diskon 10% untuk pembayaran lunas di muka!

Kami melayani seluruh area Jabodetabek tanpa biaya tambahan transportasi. Untuk area luar Jabodetabek (Bandung, Semarang, Surabaya, Bali, dll), kami juga melayani dengan biaya transportasi dan akomodasi yang akan diinformasikan saat konsultasi.

File RAW diberikan untuk Paket Pro dan Premium. Untuk Paket Basic, file RAW dapat ditambahkan dengan biaya Rp 500.000. File diberikan dalam format high-resolution JPEG dan RAW (CR2/NEF) melalui Google Drive atau Flashdisk.

Kami memiliki backup plan untuk setiap outdoor shoot. Jika cuaca tidak memungkinkan, sesi dapat dijadwal ulang tanpa biaya tambahan (reschedule gratis 1x). Atau kami bisa pindah ke lokasi indoor alternatif yang sudah kami persiapkan sebelumnya.
