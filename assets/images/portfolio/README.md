# Placeholder Images

Folder ini berisi gambar placeholder untuk portfolio website Fotoin Pictures.

## Struktur Folder
```
assets/
└── images/
    └── portfolio/
        ├── photo-1.jpg (Wedding Photography)
        ├── photo-2.jpg (Prewedding Photography)
        ├── photo-3.jpg (Product Photography)
        ├── video-1.jpg (Wedding Cinematic)
        ├── video-2.jpg (Company Profile)
        └── event-1.jpg (Corporate Event)
```

## Cara Menggunakan Gambar Anda Sendiri

1. Siapkan gambar berkualitas tinggi (minimal 1920x1080px)
2. Simpan gambar dengan nama yang sesuai di folder ini
3. Format yang direkomendasikan: JPG atau PNG
4. Ukuran file maksimal: 2MB per gambar (untuk loading yang cepat)

## Gambar yang Dibutuhkan

- **photo-1.jpg** - Foto wedding/pernikahan
- **photo-2.jpg** - Foto prewedding
- **photo-3.jpg** - Foto produk
- **video-1.jpg** - Thumbnail video cinematic
- **video-2.jpg** - Thumbnail company profile
- **event-1.jpg** - Foto event/acara

## Tips
- Gunakan gambar dengan komposisi yang menarik
- Pastikan pencahayaan baik dan fokus tajam
- Gambar horizontal (landscape) lebih cocok untuk layout
- Compress gambar untuk optimasi loading website
